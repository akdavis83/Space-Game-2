Enhanced Space Game

Game Controls

Mouse: Move to control ship direction

Space Bar or Click: Fire bullets

P: Pause/Resume game

R: Restart when game over

Enhanced Features:

  • Improved ship graphics with detailed design
  • Enhanced enemy ships with different types and behaviors
  • Proper collision detection system
  • Game menu with score tracking
  • Health system for both player and enemies
  • Multiple enemy types: Basic, Fast, Heavy, Hunter
  • Visual effects: explosions, trails, shields
  • Progressive difficulty scaling
  • Lives system and game over screen
  • Pause functionality
